# Product Overview

A full-stack Todo application built for learning and practicing application design, full-stack integration, and domain modeling.

## Core Capabilities

- Todo CRUD with per-user data isolation (userId scoping)
- Session-based authentication (register / login / logout)
- Server-side auth guard — no client-side trust
- REST API backend with rate limiting and request validation

## Target Use Cases

- Authenticated users managing personal todo lists
- Learning platform for practicing full-stack patterns (API design, auth, state management)

## Value Proposition

Demonstrates production-quality patterns (layered architecture, session auth, Docker-based deployment) in a minimal, understandable codebase — not just a UI prototype.
